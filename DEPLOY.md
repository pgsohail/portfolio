## Quick Deploy to GitHub Pages

The repository now includes a GitHub Actions workflow for automatic deployment!

### Recommended: Use GitHub Actions (Easiest)

1. Go to: https://github.com/pgsohail/portfolio/settings/pages
2. Under **"Source"**, select **"GitHub Actions"**
3. Push to `main` or `master` branch
4. Your site deploys automatically!

**Your portfolio will be live at:** https://pgsohail.github.io/portfolio/

---

## Alternative: Manual Branch Deployment

### Step 1: Enable GitHub Pages

1. Go to: https://github.com/pgsohail/portfolio/settings/pages
2. Under **"Source"**, select **"Deploy from a branch"**
3. Select **"main"** branch (or your current branch)
4. Select **"/ (root)"** folder
5. Click **"Save"**

### Step 2: Access Your Site

After a few minutes, your portfolio will be live at:
**https://pgsohail.github.io/portfolio/**
